# Proxy & Vpn scanner🔥

Developed by **yagami0xx**

Advanced Proxy & VPN Intelligence tool designed for deep analysis of proxies and VPN IP addresses.

This tool performs multi-layer inspection including WHOIS (RDAP), Geo-IP validation, DNS leak detection, classification, and SOCKS5 tunneling.

---

## 🚀 Features

- 🔎 RDAP WHOIS ASN Lookup
- 🌍 Accurate Country & Continent Detection
- 🛰 IP-API Geo Validation
- 🧠 Smart Proxy Classification (Datacenter / ISP / VPN)
- 🧪 DNS Leak Test
- ⚡ Latency Speed Measurement
- 🏳 Automatic Country Flags
- 🔌 SOCKS5 Local Tunnel (127.0.0.1:1080)
- 📂 File Scanning (Bulk Mode)
- 🎯 Single Proxy Scan Mode
